# 🎓 Smart Coaching Management System

An AI-powered Coaching Management System that helps coaching institutes manage students, teachers, attendance, fees, exams, and AI-powered learning tools through a modern and responsive web interface.

---

## 📌 Overview

Smart Coaching Management System is a modern educational platform designed to simplify coaching institute operations. It provides multiple management modules along with AI-powered features to improve learning and administration.

---

## ✨ Features

### 📚 Institute Management

- Student Management
- Teacher Management
- Parent Portal
- Attendance Tracking
- Fee Management
- Homework & Assignments
- Announcements

- Learning Resources
- Reports & Analytics
